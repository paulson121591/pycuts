## set_y ##

```python
fpdf.set_y(y: float)
```
### Description ###

Moves the current abscissa back to the left margin and sets the ordinate. If the passed value is negative, it is relative to the bottom of the page.

### Parameters ###

y:
> The value of the ordinate.
